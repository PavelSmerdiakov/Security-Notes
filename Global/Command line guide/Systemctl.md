Description :

- Démarrer, arrêter, redémarrer, recharger un service : Par exemple, systemctl start nom-du-service pour démarrer un service.
- Activer ou désactiver un service au démarrage du système : Par exemple, systemctl enable nom-du-service pour activer un service au démarrage.
- Vérifier l'état d'un service : Par exemple, systemctl status nom-du-service pour afficher l'état actuel d'un service.
- Afficher les journaux de service : Par exemple, systemctl status nom-du-service pour consulter les journaux d'un service spécifique.
- Gérer les unités systemd, telles que les cibles, les timers, les sockets, etc.
